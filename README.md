## Rock Crabs and PCA

Principal component analysis is useful in analyzing high-dimensional data. In RockCrabsParallelAxes, Parallel Axes are used to visualize multi-variates of Rock Crabs, moving away from the traditional cartesian plane. In RockCrabsRadialAxes, Radial Axes are used to visualize multi-variates of Rock Crabs, providing a tool to see if correlations between variates exist. In RockCrabsPrincipleComponents, PCA is carried out to identify the 4 groups of Rock Crabs using pairs plots.
